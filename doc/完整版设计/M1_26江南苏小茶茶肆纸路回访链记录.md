# M1.26 江南苏小茶茶肆纸路回访链记录

日期：2026-06-16

## 节点目标

M1.25 已关闭临安油纸伞与太湖云锦 / 缂丝两条内容回访链，本节点继续关闭江南剩余内容缺口中最小且最明确的一条：临安湖畔茶肆与苏小茶生活回访链。

本轮不新增美术资源、不新增地图点位、不改变正式跨区路线落点；临安仍通过街景 gate / 区内通道到达，`smokeSubregion=jiangnan-linan` 只作为 DEV 浏览器 QA 起点。

## 协作审计结论

- 按 M1 固定 6 团队口径启动只读审计，覆盖集成结构、江南内容语义、跨区旗标命名、徽州纸路衔接、结局回声和 QA smoke。
- 6 个只读团队均已完成并关闭；本轮不保留空转子智能体。
- 审计结论支持复用现有 `HomeVisitDef` 链路，不新增 reducer action：首访陈列茶席，荐藏复单交付后再要求跑过 `jn-lake-tea-house`，随后解锁纸路熟客回访和续订单。
- 命名采用 `xiaocha-paper-tea-*` 前缀，避免与黔滇茶马 `tea-road`、徽州文房 `stationery` 和岭南染织声誉旗标混淆。

## 程序落点

苏小茶人物线：

- `jn-su-xiaocha` 仍保留在 `PLACEHOLDER_NPCS`，通过 `REGIONAL_NPC_OVERRIDES` 增加 `homeVisit` 功能、茶席 / 纸路偏好、日程、纸路情报和结局影响。
- `jn-lake-tea-house` 继续复用既有活动与 `jn-lake-tea-house-listen` 挑战；活动完成会写入 `heard-huizhou-paper-route` 与 `route-known:route-jiangnan-huizhou-paper`，作为二次回访前置。

家访链：

- 新增 `homevisit-xiaocha-lake-tea-gallery`：要求珍品阁陈列 `tea`，可选择“茶席入账”生成“湖畔茶席复单”。
- 新增 `homevisit-xiaocha-paper-route-return`：要求 `homevisit-referral-completed:xiaocha-paper-tea-ledger`、`heard-huizhou-paper-route` 和 `route-known:route-jiangnan-huizhou-paper`，避免只完成荐藏复单、不跑临安茶肆活动就解锁回访。
- 回访收藏分支 `xiaocha-paper-tea-renewal-ledger` 会写入 `collector-reputation-paper-tea-renewed`，并生成“纸路茶笺续订单”，续单验收 `xuanPaper`，让茶肆生活线真正读到徽州纸路成品。
- 结局 `relationshipOutcomes` 通过 `jn-su-xiaocha` 专属分支读取 `collector-reputation-paper-tea-renewed` / `homevisit-xiaocha-paper-route-return-resolved`，追加临安茶席与纸路熟客后日谈。

章节与守卫：

- `REGION_CHAPTERS.jiangnan.orderHooks` 增加 2 个苏小茶 homeVisit hook，均标记 `readsItemState: true`。
- 江南 `homeVisitIds` 接入 2 条新家访 ID。
- `FULL_SCOPE_REGION_REQUIREMENTS.jiangnan.m1Actions` 移除“补临安茶肆与苏小茶生活回访”，剩余动作转向蓝染 / 竹编专属工艺深度、江南多入口 smokeBindings 和灯市后续单关系读数。
- 冻结守卫纳入 2 条新 homeVisit ID，避免后续改名导致存档 / 文档漂移。
