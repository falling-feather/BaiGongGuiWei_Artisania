# M1.17 江南苏杭水市与百工院人工地图记录

日期：2026-06-15

## 目标

本切片关闭江南章节 `chapter-jiangnan-baigong-homecoming` 的直接人工地图缺口：`jiangnan-suhang` 与 `jiangnan-baigongyuan`。本轮只冻结程序入口、交互目标、守卫矩阵与验证口径，不扩展临安 / 太湖长线，不处理最终美术资源、装饰密度和精细坐标摆位。

## 交互点

`jiangnan-suhang`：

- industry：`harvest-indigo`、`build-indigo`、`harvest-bamboo`、`split-bamboo`
- craft：`indigo-dyeing`、`bamboo-weaving`
- NPC：`jn-indigo-keeper`、`jn-bamboo-master`
- 区内 gate：`jiangnan-longquan`、`jiangnan-jinling`、`jiangnan-linan`、`jiangnan-taihu`、`jiangnan-baigongyuan`
- 跨区 gate：`huizhou`、`ganpo`

`jiangnan-baigongyuan`：

- craft：`indigo-dyeing`、`bamboo-weaving`
- activity：`jn-yard-fields`
- NPC：`jn-xiaoman`
- 区内 gate：`jiangnan-suhang`、`jiangnan-jinling`、`jiangnan-longquan`、`jiangnan-linan`、`jiangnan-taihu`
- 跨区 gate：无；正式跨区移动应先回苏杭或金陵路线落点。
