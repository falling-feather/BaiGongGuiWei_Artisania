# M1.23 荆楚铜铁矿场与湘绣楼人工地图记录

日期：2026-06-16

## 本轮目标

M1.23 关闭荆楚章节最后两个直接人工地图断点：`jingchu-mine-yard` 与 `jingchu-xiang-embroidery`。本轮只处理程序可达、地图入口、活动钩子、冻结守卫和浏览器验证口径，不把荆楚章节直接升级为完成态。

正式移动继续遵守街景 gate / 区内通道原则：跨区路线 `route-bashu-jingchu-river`、`route-qiandian-jingchu-mine`、`route-jingchu-ganpo-lake` 仍落到 `jingchu-lake-market`，不把矿场或湘绣楼改成新路线默认落点。`smokeSubregion` 只作为 DEV-only 浏览器 QA 起点。

## 程序落点

1. 新增 `src/data/mapLayouts/jingchu-mine-yard.json`，并注册进 `RUNTIME_MAP_LAYOUTS`。
   - 地图落入采铜矿、采铁矿、炼铜、冶铁、`jc-daye-mine` 活动、`jc-yeshu` NPC。
   - 区内 gate 连接江湖渡市、楚漆坊、湘绣楼；跨区 gate 保留巴蜀、黔滇、赣鄱方向。
2. 新增 `src/data/mapLayouts/jingchu-xiang-embroidery.json`，并注册进 `RUNTIME_MAP_LAYOUTS`。
   - 地图落入采蚕茧、缫丝、湘绣地图工艺入口、`jc-xiang-embroidery` 活动、`jc-wen-xiuniang` NPC。
   - 区内 gate 连接江湖渡市、楚漆坊、铜铁矿场；跨区 gate 保留巴蜀、黔滇、赣鄱方向。
3. 新增两个活动挑战：
   - `jc-daye-mine-vein`：承接大冶矿场分矿色、配炉口和护矿路的最小可玩钩子。
   - `jc-xiang-embroidery-needle`：承接湘绣定明暗、鬅毛针收神和绣样回样的最小可玩钩子。
4. 地图、章节、完整版审计、冻结 ID、街景 spec、导航入口和 DEV 起点测试全部纳入 M1.23 守卫。

## 冻结边界

- 新增冻结子区 / layout：`jingchu-mine-yard`、`jingchu-xiang-embroidery`。
- 新增冻结活动：`jc-daye-mine`、`jc-xiang-embroidery`。
- 新增冻结活动挑战：`jc-daye-mine-vein`、`jc-xiang-embroidery-needle`。
- 新增冻结 NPC：`jc-yeshu`、`jc-wen-xiuniang`。
- 新增冻结产业 / 地图工艺入口：`harvest-copper-ore`、`harvest-iron-ore`、`smelt-copper`、`smelt-iron`、`harvest-cocoon`、`sericulture`、`xiang-embroidery`。

`xiang-embroidery` 本轮只作为地图工艺入口冻结，不冻结专属 `CraftInteractionSpec`。后续补湘绣三阶段工艺、缺陷、返修、绣样订单差异时，再把它纳入专属工艺交互冻结清单。

## 未关闭内容

- 荆楚仍保持 `needs-expansion`：后续要补渡口水运三阶段摊位、楚漆修复后续追单、矿料状态反馈、湘绣楼人物回访和绣样订单差异。
- 大冶矿场目前具备铜铁料入口与活动挑战，但未做矿料读数、矿证回访和铜铁订单回流。
- 湘绣楼目前具备地图工艺入口与活动挑战，但未做专属工艺交互、绣样缺陷读取和人物长线。
- M1.23 截止时全项目剩余未入轨人工地图收敛为 `jiangnan-linan`、`jiangnan-taihu`；该缺口已在后续 M1.24 关闭。
- 最终美术资源、地形细化、NPC 站姿、装饰层和精确点位继续后置。
