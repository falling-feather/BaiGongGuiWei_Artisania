# 完整版设计档案

本目录保存 `ed7d2cf` 快照时项目的完整规划口径，用于长期愿景、后续扩展和跨组参考。这里的文档不代表当前冲刺必须全部完成；当前冲刺以 `doc/当前优先设计` 为执行准线。

## 使用原则

1. 完整版设计保留 11 个大地区、43 个小地区、全地区工艺、长期 NPC、节令、商路、美术与联网路线。
2. 新增系统、剧情或美术需求如果超出当前优先版，应先记录在完整版档案或原文档，不直接进入当前冲刺。
3. 当前优先版完成后，再从本目录回收下一批地区、人物线和工艺深度。
