# M1.24 江南临安水市 / 太湖织埠人工地图记录

日期：2026-06-16

## 节点目标

本节点关闭完整版范围内最后两张未接入运行时的人工地图：`jiangnan-linan` 与 `jiangnan-taihu`。M1.17 关闭的是江南章节入口图，本轮关闭的是 43 个小地区全量人工地图的最后缺口。

本轮不新增江南章节大口径，不扩出新的路线或工艺规格；重点是把已有临安茶伞、太湖缂丝、湖港生活、NPC 与街景 gate 落到可运行、可测试、可浏览器验收的程序链路里。

## 协作审计结论

- 地图与测试：`RUNTIME_MAP_EDITOR_SNAPSHOTS`、`mapLayout.test.ts`、`priorityMapLayoutEntrypoints.test.ts`、章节入口与 full-scope 审计需要同步从 41/43 改为 43/43。
- 内容边界：临安/太湖已有 `SUBREGION_CONTENT`、活动、NPC、工艺交互，不需要新开工艺 ID；本轮只补人工地图落位。
- 路线红线：`route-jiangnan-huizhou-paper`、`route-jiangnan-ganpo-kiln` 仍落江南 `jiangnan-suhang`；`route-jiangnan-jingji-canal` 仍落 `jiangnan-jinling`。临安/太湖可以有街景 gate，但不得改正式 route landing。
- 冻结边界：新增冻结的是小地区/layout、临安/太湖活动和 NPC；`oilpaper-umbrella`、`kesi` 已是冻结 craft，不新增重复 craft 冻结。
- 文档口径：M1.24 后不再记录临安/太湖人工 JSON 缺口；后续缺口转为临安茶肆/伞铺雨季回访、太湖缂丝/云锦藏客订单差异、蓝染/竹编深度和多入口 smokeBindings。
