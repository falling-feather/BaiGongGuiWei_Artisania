# M1.25 江南临安 / 太湖内容回访链记录

日期：2026-06-16

## 节点目标

M1.24 已关闭临安水市与太湖织埠的人工地图断点，本节点继续关闭其中最明确的内容端缺口：临安油纸伞雨季荐藏回访链，以及太湖云锦 / 缂丝藏客订单差异链。

本轮不新增美术资源、不改正式跨区路线落点、不扩大江南章节大口径。正式移动仍只通过街景 gate / 交互点触发；`smokeSubregion` 只作为 DEV 浏览器 QA 起点。

## 协作审计结论

- 按 M1 固定 6 团队口径启动只读审计，覆盖江南章节绑定、临安雨伞回访、太湖织样回访、订单钩子、冻结 ID 与 QA 风险。
- 6 个只读团队均已完成并关闭；本轮不再保留空转子智能体。
- 审计结论支持把 M1.24 后续缺口先拆成“小而闭环”的内容端链路：初访看作品、荐藏订单、活动验样旗标、二次回访、续订订单、结局后日谈。
