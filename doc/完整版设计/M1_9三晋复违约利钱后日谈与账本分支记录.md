# M1.9 三晋复违约利钱后日谈与账本分支记录

日期：2026-06-15

## 节点目标

M1.9 关闭 M1 后续动作中“三晋复违约利钱后日谈与更多账本分支”的第一轮程序闭环。目标不是把三晋升级为 `chapter-ready`，也不补煤铁窑塬 / 清徐醋坊人工地图，更不新增 M3 级债务账本字段；本轮只把“旧失约已补后再次违约”的利钱挂账、雷掌柜票号账本后日谈、补账续单和结局回声接成可测试链路。

执行依据：[程序大型化协作指引.md](程序大型化协作指引.md)、[M1全地区核心章节入库记录.md](M1全地区核心章节入库记录.md) 与 [M1_8三晋信用违约回本修复切片记录.md](M1_8三晋信用违约回本修复切片记录.md)。

## 本轮程序落点

1. `src/engine/reducer.ts`
   - 雷掌柜信用单在“旧失约已由漆柜续单修复后”再次过期时，写入 `credit-repeat-default:sj-lei-zhanggui`、`sanjin-credit-interest-accrued`、`sanjin-credit-ledger-aftertalk-open`。
   - `orderTrustScore` 与雷掌柜信用文案读取 `sanjin-credit-interest-accrued` 且未 `sanjin-credit-interest-settled` 时，进一步压低信用并显示“复违约利钱入账”。
   - 完成雷掌柜后续信用单或 `lei-compound-ledger-order` 补账续单后，写入 `sanjin-credit-interest-settled`、`sanjin-credit-ledger-aftertalk-settled`、`credit-repeat-default-settled:sj-lei-zhanggui`。
   - 雷掌柜结局后日谈读取未补、已补、复违约利钱未清和已清账四类状态。
2. `src/data/regionContent.ts`
   - 雷掌柜新增 `homeVisit` 功能。
   - 新增 `homevisit-lei-credit-ledger-aftertalk`：需要 `sanjin-credit-ledger-aftertalk-open`，只被 `sanjin-credit-ledger-aftertalk-settled` 阻断；活跃补账续单的重复生成由 reducer 订单生成层防重，过期后仍可回到留凭分支清账。
   - 三个分支分别为 `lei-interest-receipt` 直接留凭结清、`lei-compound-ledger-order` 生成“票号补账押货续单”、`lei-coal-vinegar-ledger-clue` 写煤铁 / 醋坊后续账线线索。
3. `src/data/regionChapters.ts`
   - 三晋章节纳入 `homevisit-lei-credit-ledger-aftertalk` 的 `orderHooks` 与 `homeVisitIds`。
   - 三晋 gap 从“仍缺复违约利钱后日谈和更多账本分支”改为“已有复违约利钱挂账与票号账本后日谈；仍缺 M3 级多轮本金 / 利钱统计和煤铁 / 醋坊账线实化”。
4. `src/data/__tests__/priorityFrozenIds.test.ts`
   - 冻结 `homevisit-lei-credit-ledger-aftertalk`。
5. `src/engine/__tests__/sanjinCreditRecovery.test.ts`
   - 扩展专项测试到复违约利钱、后续信用条款、雷掌柜账本后日谈、补账续单、直接留凭结清、重复刷单阻断、补账续单过期后回到留凭分支和结局回声。

## 当前限制

- 本轮不新增 `ActiveOrder` 债务字段，不统计本金、利钱金额、违约次数或复利；这些属于 M3 长线经济。
- `sj-piaohao` 活动生成订单继续保持 `route`，不改成 `credit`，避免破坏活动订单语义。
- `lei-coal-vinegar-ledger-clue` 只写煤铁 / 醋坊账线线索，真实煤铁重货保票与清徐醋坊民生日用账线留给 M1.10 或后续节点。
- `sanjin-coal-yard` 与 `sanjin-vinegar-yard` 仍缺人工 JSON；正式移动继续通过街景 gate/通道触发，不通过总览地图迁移。

## 当前判断

M1.9 让三晋票号从“旧失约可补回”推进为“补回后再次违约会产生利钱挂账，并能通过雷掌柜账本后日谈或补账续单收束”的可测试章节链。三晋仍不是完整章节：后续更适合补煤铁 / 醋坊账线实化、煤铁窑塬 / 清徐醋坊人工地图、人物回访和 M3 多轮经济，而不是在本轮提前做全量金融账本。
