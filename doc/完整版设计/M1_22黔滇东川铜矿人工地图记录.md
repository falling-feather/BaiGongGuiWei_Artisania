# M1.22 黔滇东川铜矿人工地图记录

日期：2026-06-16

## 1. 本轮目标

M1.22 只关闭黔滇章节的直接人工地图断点：`qiandian-dongchuan-copper`。本轮不扩写东川矿口回访、铜银材料状态反馈、蜡染人物长线、岭南港口回访、建水陶专属工艺规格或路线默认落点。

正式移动仍以街景 gate / 区内通道触发为准；`smokeSubregion` 仅用于 DEV 浏览器 QA 起点，不作为正式迁移方式。

## 3. 程序落点

- 新增 `src/data/mapLayouts/qiandian-dongchuan-copper.json`，注册进运行时 layout 管线。
- 东川铜矿落入 `harvest-copper-ore`、`smelt-copper`、`jianshui-pottery`、`qd-dongchuan-mine` 与 `qd-tongshan-ke`。
- 街景 gate 连接 `qiandian-miao-village`、`qiandian-tea-road`，并提供通往巴蜀、岭南、荆楚的跨区出口。
- 新增活动挑战 `qd-dongchuan-mine-vein`，让 `qd-dongchuan-mine` 从地图入口推进到可执行的最小玩法钩子。
- `REGION_CHAPTERS` 的黔滇入口小地区扩为苗寨、茶马驿道、东川铜矿三处；章节状态仍为 `needs-expansion`。

## 4. 冻结边界

本轮冻结以下程序入口目标：

- 子区与 layout：`qiandian-dongchuan-copper`
- 活动：`qd-dongchuan-mine`
- 活动挑战：`qd-dongchuan-mine-vein`
- NPC：`qd-tongshan-ke`
- 产业入口：`harvest-copper-ore`、`smelt-copper`
- 地图工艺入口：`jianshui-pottery`
- 街景连接目标：`qiandian-miao-village`、`qiandian-tea-road`、`bashu`、`lingnan`、`jingchu`

最终矿墙、矿车、洗矿槽、炉棚、陶棚、NPC 站姿和碰撞精调仍可交给后续美术替换与点位调整；替换时必须保留 industry / craft / activity / npc / gate 的目标 ID。

## 5. 未关闭内容

- 东川矿口回访与铜银材料状态反馈。
- 蜡染人物长线回访与银染互证后日谈。
- 茶马长线延展到岭南港口的回访与订单差异。
- `jianshui-pottery` 专属三阶段工艺交互、缺陷、返修与订单反馈。
- 黔滇跨区路线默认落点调整。本轮明确不调整 route landing。
