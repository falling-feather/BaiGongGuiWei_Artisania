# M1.4 真实订单链补线记录

日期：2026-06-14

## 节点目标

M1.4 关闭 M1 后续动作中的“第一批 needs-expansion 章节真实订单链”项。目标不是把六个章节一次性升为 `chapter-ready`，而是先让黔滇、荆楚、徽州、京畿、三晋、雪域的章节入口活动都能生成真实 `ActiveOrder`，并让交付验收读取同资源 `ItemInstance` 的品相。

执行依据：[程序大型化协作指引.md](程序大型化协作指引.md)、[M1全地区核心章节入库记录.md](M1全地区核心章节入库记录.md) 与 [M1_3独立章节Smoke入口记录.md](M1_3独立章节Smoke入口记录.md)。

## 当前限制

本轮关闭的是“第一批真实活动订单入口”，不是六章的最终内容完成。

仍未关闭：

- 黔滇苗银礼俗家访、协作配方、银染节事三阶段摊位。
- 荆楚渡口水运三阶段摊位、楚漆修复更长回访、湘绣楼人物线。
- 徽州商会信用风险、纸墨荐藏回访扩展、墨砚深巷 / 歙石山坑人工地图。
- 京畿采办许可读取地区声望与商誉、宫造采办长线分支、都门市口人工地图。
- 三晋票号信用的长期违约 / 回本指标、煤铁窑塬 / 清徐醋坊人工地图。
- 雪域高原供展回访、雪山驿口 / 颜料矿谷 / 银器帐房人工地图。
- 多入口 `smokeBindings` 仍未实装。

M1 的内容缺口从“六个 needs-expansion 章节缺真实订单链”推进为“六个 needs-expansion 章节已有活动即时订单，但仍缺长线章节化内容”。后续 M1.5 优先建议从黔滇节事摊位 / 家访 / 协作、京畿声望采办、三晋信用违约、雪域唐卡材料状态回访四类长线系统中选最能支撑美术交接的一组继续收束。
