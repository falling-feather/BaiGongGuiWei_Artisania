# 百工归位 · Artisania — 美术资源设计清单

> 本文目标：把当前 demo 里**所有需要美术的子窗口与画面元素**逐一列清，供绘制像素 / 国风美术时按图索骥。
> 读完本文你应当能回答：游戏一共有几个窗口、每个窗口要画哪些图、每张图的尺寸 / 风格 / 数量、当前用什么占位、替换到哪里。
>
> **总体风格基调**：中国非遗题材 · 国风水墨 + 像素质感；主色取自 `src/index.css` 的 `:root` 调色板
> （靛蓝 `--indigo`、竹绿 `--bamboo`、朱砂、纸白 `--paper`、墨褐线条 `--line`）。
> **像素世界单格** `TILE = 32px`（见 `src/game/textures.ts`），UI 弹窗为常规网页矢量/位图皆可。
>
> **替换原则**：玩法零改动。Phaser 贴图只动 `game/textures.ts`（占位）→ 改为 `scene.load.*` 加载真实图；
> UI 窗口美术多为 CSS 背景 / `<img>` 占位块，替换图片资源即可。真实素材建议放 `public/assets/`。

## 1. 调色与风格规范（绘制前先读）

- **像素世界**：`TILE=32px`。地砖/路面/水面为 1×1 格；建筑（作坊/牌坊）为 2×2 格（64×64）；
  角色/NPC 为 1 格内的小人（约 16~24px 居中，留脚部阴影空间）。像素硬边、低饱和国风配色。
- **UI 弹窗**：跟随 `index.css` 既有视觉（纸白底、墨褐描边、朱砂强调、靛蓝标题）。图标建议 24~48px、
  描边风或单色剪影，避免与像素世界违和。
- **地区配色驱动**：地面/街道色由地区 `terrain.palette` 前两色运行时 `setTint`，所以**地砖底图请画成白/浅灰**
  （`TEX.ground` 即白底），让着色生效；不要把颜色画死。

## 2. 游戏世界（Phaser `StreetScene`）——核心像素资源

> 全部在 `src/game/textures.ts` 以 `Graphics.generateTexture` 程序生成占位。替换时改为
> `scene.load.spritesheet/image(key, url, ...)`（在 `StreetScene.preload()`），`TEX` 键不变即可。

### 2.1 地面 / 地貌瓦片（1×1 格 = 32×32）

| 纹理键 `TEX.*` | 用途 | 占位 | 待绘要点 |
|---|---|---|---|
| `ground` | 通用地面（草地/泥土） | 白底+噪点 | **画成白/浅灰**供 tint 着色；像素草纹 |
| `road` | 街道路面 | 土黄 | 石板/夯土路，可与 ground 区分 |
| `wall` | 墙体 | 棕 | 砖墙/夯土墙 |
| `water` | 河面 / 海面（**阻挡**） | 蓝波纹 | 江南河水，波光；不可通行 |
| `bridge` | 石桥（**可通行**，横跨河面街道行） | 米褐桥板 | 石拱桥板，与 water 接缝自然 |
| `fence` | 栅栏（地块边界·**阻挡**） | 木栏 | 竹/木栅栏 |
| `tree` | 树木（点缀·**阻挡**） | 绿冠+棕干 | 江南树/松，带阴影 |
| `rock` | 山石（点缀·**阻挡**） | 灰三角 | 太湖石/山岩 |

> **地貌类型** `TerrainKind`（`water`/`mountain`/`coast`/`plain`）决定铺哪些瓦片，见开发者文档 §4.8 模块 1。
> 江南为 `water`：竖向河流 + 街道行石桥。未来每地区可换不同地貌瓦片集。

### 2.2 建筑 / 交互点（2×2 格 = 64×64）

| 纹理键 | 用途 | 占位 | 待绘要点 |
|---|---|---|---|
| `shopIndigo` / `craft` | 手艺作坊（靛蓝顶） | 蓝屋 | 中式坊门小楼，可按手艺主题变体 |
| `shopBamboo` | 竹编坊（竹绿顶） | 绿屋 | 竹构作坊 |
| `indHarvest` | 采集业点（绿） | 绿屋 | 田园/采集棚 |
| `indRefine` | 精炼业点（橙·炉火） | 橙屋 | 炉窑/作坊，有火光 |
| `indProduct` | 制作业点（紫·精品） | 紫屋 | 精工坊 |
| `gate` | 地区出入口·牌坊 | 朱柱+匾 | 中式牌坊，匾额可留白写地名 |

> **进阶**：每个手艺/产业点可拥有专属建筑图（按 `texture` 键扩展），让街市更有辨识度。

### 2.3 角色 / NPC（1 格内小人）

| 纹理键 | 用途 | 占位 | 待绘要点 |
|---|---|---|---|
| `player` | 玩家匠人 | 朱衣小人 | **建议做四向行走精灵表**（up/down/left/right，每向 2~4 帧），替换后在 `StreetScene` 建 `anims` |
| `npcTourist` | 游客 NPC（街市游走） | 青衫小人 | 可做多套服色区分不同游客 |
| `npcVendor` | 关联人物 NPC（驻守手艺点） | 赭衣笠帽 | 篾匠/染坊主等，宜有职业辨识度 |

### 2.4 标记 / 特效

| 纹理键 | 用途 | 占位 | 待绘要点 |
|---|---|---|---|
| `marker` | 「按 E」交互提示气泡 | 黄三角 | 气泡/箭头，可加动画帧 |
