# 主轴街景导航目标记录

更新时间：2026-06-14

## 本轮新增

`src/game/regionSpec.ts` 现在会在没有手动追踪百科词条时，把当前优先版主轴目标转换为街景 `navigationTarget`。

这意味着：

1. 玩家没有追踪百科时，街景会默认高亮主轴下一步的区内通道或跨区 gate。
2. 玩家手动追踪百科词条时，百科行脚目标优先，主轴目标只作为 fallback。
3. 高亮目标仍然只用于提示和小地图标记，不触发迁移。
4. 正式迁移仍由街景内 `gate` / `subregionGate` 交互派发，世界地图不承担直接迁移。

## 已加测试

更新 `src/game/__tests__/regionSpec.test.ts`：

- 无百科追踪时，初始江南会指向 `jiangnan-longquan`。
- 手动追踪百科时，会覆盖主轴 fallback。
- 江南主轴闭环后，下一步会指向通往后续路线的街景 gate。
