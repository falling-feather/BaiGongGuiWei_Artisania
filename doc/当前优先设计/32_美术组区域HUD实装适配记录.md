# 美术组区域 HUD 实装适配记录

更新日期：2026-06-17

## 本轮目标

- 解决顶栏 UI 过于像“素材硬贴”的问题，将常用入口、当前位置、天气时间、设置合并为统一的游戏 HUD 材质带。
- 让顶栏、侧边任务栏、底部交互提示共享地区主题色，避免不同地区 UI 气质割裂。
- 保留已经生成的像素 UI 素材，但降低其作为整块贴图的存在感，改为边框纹理、按钮纹理和材质细节。

## 实装范围

- `src/components/Hud.tsx`
  - 接入 `minigameRegionTheme(currentRegion)`。
  - 为顶栏、任务侧栏、交互提示、首启操作提示注入 `--hud-accent` 与 `--hud-accent-soft`。
  - 顶栏增加统一底板图层 `hud__top-frame-v2`，原导航底图退为隐藏兜底，不再单独撑起一块大贴图。

- `src/index.css`
  - 新增 HUD art direction pass。
  - 顶栏改为单条半透明材质带，桌面端三列布局：工具入口 / 地点时间 / 设置。
  - 移动端改为两行：状态 + 设置在上，工具按钮横向滚动在下。
  - 任务侧栏改为较轻的任务栏样式，使用地区色强调当前目标。
  - 底部交互提示与一次性操作提示同步地区色和材质边框。

## 地区主题继承

当前沿用 `src/components/minigameUiTheme.ts` 的五类主轴主题：

| 主题 | 直接地区 | 继承地区 | HUD 色调 |
|---|---|---|---|
| 江南 | `jiangnan` | `huizhou` | 水青、竹色 |
| 巴蜀 | `bashu` | `qiandian` | 茶棕、青铜 |
| 岭南 | `lingnan` | `jingchu` | 商埠绿、石雕灰 |
| 赣鄱 | `ganpo` | `jingji` | 瓷蓝、窑火暖灰 |
| 西域 | `xiyu` | `western`, `sanjin`, `xueyu` | 驼铃金、织毯暖色 |

后续新增地区 UI 不需要重新写 HUD 结构，只要补充或调整 `minigameUiTheme.ts` 的主题映射。
