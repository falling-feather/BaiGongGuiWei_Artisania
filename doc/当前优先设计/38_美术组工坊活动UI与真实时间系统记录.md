# 美术组工坊活动 UI 与真实时间系统记录

日期：2026-06-17

## 本轮目标

- 将工坊整页、产业整页、活动弹窗和 NPC 交流页继续向当前像素美术风格靠拢。
- 使用 `image_gen` 生成的像素 UI 图集封装地区框、玩法圆标与计时条。
- 给游戏接入真实时间限制：采集资源等待 10 分钟、加工等待 3 分钟、工坊制作等待 2 分钟、地区活动等待 5 分钟、同一 NPC 每个真实日期只能普通攀谈一次。

## 规则接入

- 新增 `src/engine/realTime.ts`。
- `GameState` 新增 `realTime`，存档版本升级到 18。
- `src/store/gameStore.ts` 对 `RUN_PROCESS`、`GATHER_RESOURCE`、`PERFORM_ACTIVITY`、`TALK_NPC` 自动注入 `Date.now()`。
- reducer 仅在 action 带 `now` 时启用真实冷却，避免旧测试和开发脚本被误阻断。

## 后续注意

- 如果后续继续调用 `image_gen` 生成全屏大框，可替换当前 `workshop_region_*.png` 为更完整的 16:9 页面框；当前版本已可作为统一风格过渡资源。
- 地区活动和工坊页面已经共用同一套工坊 UI 主题，后续新增玩法应优先通过 `minigameUiTheme.ts` 注册资产，避免直接写死图片路径。
