# 美术组 UI 对齐与瓦片细化记录

日期：2026-06-17

## 本轮目标

- 继续推进 NPC 二级交互页的 UI 对齐与风格统一。
- 降低江南地图地面与庭院瓦片的碎裂感、棋盘感。
- 保持当前 imagegen 像素资源来源可追踪，优先处理已落盘资源与运行时铺设规则。

## 瓦片地图优化

- 处理脚本：`tools/package_region_terrain_tiles.py`
- 输出目录：`public/assets/game/terrain/regions/<region>/`
- Manifest：`public/assets/art_sources/priority_art/2026-06-14/05_manifests/region_terrain_tiles_manifest.json`
- 瓦片预览：`public/assets/art_sources/priority_art/2026-06-14/05_manifests/region_terrain_tiles_preview.png`

本轮将地区地貌瓦片的后处理从单纯降噪推进到边缘融合：地面、道路、水岸、庭院等瓦片在量化前增加平均色边缘柔化，减少 32x32 拼接时的硬接缝。地面与庭院瓦片的细节保留比例下调，让它们更像连续铺面，而不是每格都有强独立纹样。

运行时同步调整 `StreetScene` 铺设：庭院瓦片只在道路交汇与近路带承担结构提示，随机 `groundAlt` 的密度降低；平原街巷的路边边界不再大面积使用庭院砖，改用柔和异色地面和少量植被，降低满屏方块重复。

## NPC 二级交互页细化

- 接入文件：`src/components/NpcDialogModal.tsx`
- 样式文件：`src/index.css`
- 统一面板：继续复用 `hud_v2_dialogue_panel.png` 图像框。
- 本轮调整：二级页内容栏增加稳定左内距，避免窄屏下标题与标签被中间木框分隔线压住；页签卡、交流卡、委托卡和底部按钮改为同一套纸面渐变、内描边与轻阴影。

这轮重点不是再改页面结构，而是把图像框、NPC 半身像、右侧信息、四个功能入口和底部操作拉到同一视觉层级。采买夫人游客页已用于窄屏验收，半身像落位正常，右侧文本、按钮与卡片没有明显贴边或溢出。
