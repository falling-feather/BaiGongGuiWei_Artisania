# N1 主轴程序补线记录

更新时间：2026-06-14

## 节点口径

本轮承接 `23_N0基线锁定与团队分配记录.md` 的 P1-P6 审计结果，进入 `N1 主轴五区深流程` 的第一批程序补线。目标不是扩展新地区或新玩法，而是把已经纳入当前优先版的主轴入口、里程碑、点位和守卫固定下来，为后续浏览器深烟测和美术点位替换提供稳定底座。

## 已完成程序补线

1. 主轴活动目标小地区补 shipped runtime layout：
   - 江南：`jiangnan-jinling`，固定秦淮灯市、缂丝、乔照夜/宁辞秋、金陵到龙泉/百工院及跨区 gate。
   - 巴蜀：`bashu-jinli`、`bashu-tea-horse`，固定蜀锦工坊、锦官织楼、茶马会、卓锦娘/马帮阿越和山路 gate。
   - 岭南：`lingnan-harbor`，固定珠江货栈、骑楼夜市、伍海潮和黔滇商港 gate。
   - 西域：`xiyu-bazaar`、`xiyu-caravan-post`，固定绿洲巴扎、胡商萨里、驼队驿站、驼铃叔和雪域商路 gate。

2. 主轴范围定义已同步：
   - `PRIORITY_SCOPE_REQUIREMENTS` 的主轴 layout 要求从“首批入口图”扩展到“主轴实际使用小地区图”。
   - 西域主轴 NPC 要求补入 `xu-sali`，防止绿洲巴扎活动 NPC 被遗漏。

3. 西域主轴新增正式收束里程碑：
   - `journey-xiyu-caravan-route` 使用已有 `xiyu-caravan-post`，要求玩家在驼队驿站完成远行议路。
   - 这是复用现有活动/任务/路线，不新增地区、活动或迁移口径。

4. 巴蜀竹编补入工艺交互模拟：
   - 新增 `interaction-qingshen-bamboo`，覆盖选竹顺纹、破篾分丝、起底收口三阶段。
   - 补两个缺陷与两个返修项：炸篾/松口、顺纹修篾/回压收口。
   - 竹编从“仅有工艺 ID 与地图入口”提升为可被 CraftPage 和工艺模拟系统消费的内容。
