# 当前优先设计

本目录是从完整设计中裁剪出的当前执行口径。目标不是缩小世界观，而是把非美术工作压到约当前已完成工作量的 0.5 到 1 倍内，先交出一条可完整游玩的类端游生活模拟纵切，方便尽快对接美术组。

## 当前总目标

在不依赖最终美术资源的前提下，完成以下可验收状态：

1. 玩家能从新档进入江南，靠街景通道而不是总览地图完成区内和跨区移动。
2. 玩家能完成采集、精炼、制作、缺陷、返修、鉴评、售卖、订单、节令活动、NPC 见闻、商路风险、代表作和结局报告的闭环。
3. 五条主轴地区做到内容密度优先完成：江南、巴蜀、岭南、赣鄱、西域。
4. 其余六区保留可进入骨架、代表工艺、NPC/活动入口、资料卡和结局回声，但不在本轮追求端游级密度。
5. 美术组可以按稳定 ID、地区包和资产清单开工，不需要等待代码再反复改命名。

## 红线

- 地区迁移必须通过街景通道、地区入口或明确交互点；总览地图只保留调试和路线提示能力。
- 工艺交互必须继续接材料、工坊、NPC、缺陷、返修、订单和作品履历，不能退化成孤立小游戏。
- 新增内容必须至少有一个运行入口和一条测试或守卫，不能只停留在 Markdown。
- 当前优先版只冻结美术 key 和占位接入，不做最终资源替换与点位美化。
