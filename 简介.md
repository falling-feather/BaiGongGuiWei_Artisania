**《百工归位》：非遗生态经营策略游戏**

英文名：**Artisania**

**类型：** Web 端策略经营 + 卡牌模拟

**关键词：** 非遗、手艺人、传承、商业化、现代生活

玩家不是“保护一个非遗项目”，而是经营一整个“百工镇”。

镇上有木匠、染坊、陶工、皮影师、竹编师、银匠、纸伞师、修谱人、香料师等。每一种手艺都有自己的生态链：

| **手艺** | **依赖资源** | **传承风险**       |
| -------------- | ------------------ | ------------------------ |
| 蓝染           | 植物染料、水、布料 | 原料消失、审美老化       |
| 皮影           | 皮革、雕刻、唱腔   | 年轻人不学、演出场景消失 |
| 陶艺           | 土、窑、火候       | 量产替代、工艺断层       |
| 木作           | 木料、榫卯、师徒制 | 市场小、学习周期长       |

玩家每回合要决定：

1. 培养哪位学徒。
2. 把哪门手艺改造成现代产品。
3. 是否接商业订单。
4. 是否牺牲传统工序换取销量。
5. 是否举办展演、课程、联名、旅游体验。

**核心冲突：**

非遗不是“放进博物馆就保住了”，也不是“做成爆款就成功了”。

游戏里有四个数值：

| **数值** | **太低会怎样** | **太高会怎样** |
| -------------- | -------------------- | -------------------- |
| 传承度         | 技艺断代             | 过度保守             |
| 市场度         | 活不下去             | 变成廉价商品         |
| 生活度         | 离普通人太远         | 被日常消费稀释       |
| 精神度         | 失去文化内核         | 变成空洞仪式         |

**耐玩性来自：**

* 不同手艺之间可以组合，比如“竹编 + 灯具”“蓝染 + 可穿戴设备”“皮影 + 数字剧场”。
* 市场事件随机变化。
* 学徒有不同性格。
* 每次结局都生成不同的“百工镇命运报告”。

**高级之处：**

它把“非遗传承”做成一个复杂系统，而不是做成宣传片。
