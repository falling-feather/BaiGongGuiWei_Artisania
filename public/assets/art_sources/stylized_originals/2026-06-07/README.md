# 2026-06-07 风格化整图母版

本目录用于存放“先于像素化”的整张地图与角色风格原图。这里的文件不是瓦片表，也不是最终运行 PNG；它们是当前已完成地图代码的整体美术母版，用来统一土地、水系、街道、建筑、人物、季节和氛围，再向运行时素材、地图编辑器素材与后续像素化拆解。

## 本轮原则

- 先做整张土地的协调性：河道、路网、建筑群、码头、市集、作坊、前景植被和远景雾气要像同一块地方长出来。
- 再拆运行部件：从整图母版反推瓦片、建筑、道具、角色和天气层，而不是先画孤立瓦片再强行拼地图。
- 对齐现有代码：`StreetScene` 当前已经有道路网、地貌、交互点、NPC、天气/季节层和寻路逻辑；美术要覆盖这些代码已经能表现、但还缺统一设计的全部部分。
- 保留地图编辑器价值：编辑器继续负责微调落位、碰撞、交互和导出；整图母版负责统一风格、比例和空间叙事。

## 当前地图代码需要的整图美术覆盖面

| 代码部分 | 需要的整图设计对象 | 运行拆解方向 |
|---|---|---|
| 地貌 `water / mountain / coast / plain` | 每种大地貌的整体地势、水陆关系、边界过渡 | 地面、水面、水岸、山石、海岸、栅栏/边界素材 |
| 多街道网络 | 主街、横街、纵巷、十字口、桥头集散点 | 道路、竖路、十字路、桥面、湿路/雪路/落叶路 |
| 交互点 `industry / craft / activity / gate` | 建筑群落和功能区，不只是单体房屋 | 手艺坊、产业建筑、活动标记、地区出入口 |
| 地形装饰 | 河港、桥体、码头、船、柳树、茶摊、摊具、告示牌 | 道具 PNG、前景遮挡层、可/不可通行脚印 |
| NPC 与行人 | 玩家、关联人物、游客/货郎的体型与服饰体系 | 48x48 行走表、头像/立绘、职业变体 |
| 季节与天气 | 同一张土地在夏、雨、秋、冬的整体气氛 | 建筑变体、屋顶叠层、地表变体、VFX |

## 像素预览

像素预览使用外部本地工具 `D:/代码玩具测试/codex专用学术垃圾区/game_UI/game-ui-asset-forge`，通过本项目脚本 `tools/pixelize_stylized_originals.mjs` 调用。该脚本不修改外部工具目录，也不覆盖母版源图；输出都进入 `pixel_previews/`。

运行：

```bash
node tools/pixelize_stylized_originals.mjs
```

当前预览：

| 文件 | 结论 |
|---|---|
| `pixel_previews/jiangnan_full_map_pixel_preview_v01.png` | 能保住整张江南水网、桥、市集和建筑群关系；作为运行地图总构图预览可用，但进入玩法前还要手工清晰化动线和重点建筑 |
| `pixel_previews/jiangnan_building_pixel_preview_v01.png` | 当前最适合推进拆解的预览；屋顶、白墙、石桥、摊棚和作坊轮廓都比较稳 |
| `pixel_previews/core_character_lineup_pixel_preview_v01.png` | 服饰和体型方向可用；最终 48x48 行走表仍需单独清理脸、手、衣缘和动作帧 |

参数和 QA 见 `pixel_previews/manifest.json`、`pixel_previews/*.recipe.json` 与 `pixel_previews/*_qa.json`。整图 QA 的 `hasAlpha` warning 可接受，因为整张地图预览本来就是满幅背景；建筑和人物预览已通过当前工具 QA。

## 当前清单

### 地图与区域

当前大地图数据有 11 个大区：江南、西域、雪域高原、巴蜀、黔滇、三晋、京畿、荆楚、赣鄱、徽州、岭南。第一批先把江南做成完整母版，因为运行体验、NPC、手艺、主菜单封面和编辑器素材都以江南为核心。

江南小地区包含：苏杭水市、金陵城、临安水市、龙泉山坊、太湖织埠、城郊百工院。母版需要覆盖它们共同的江南水网气质，同时在后续拆分时能区分：水市、灯市、湖港、山坊、织埠、家园。

### 建筑与地标

当前运行建筑已经覆盖：蓝染铺、竹编铺、通用作坊、染坊、竹作、陶坊、金工、织造、纸坊、漆器、采集产业、精炼产业、成品产业、田作、矿采、窑火、锻造、织机、市集、地区牌坊。

开放世界建筑已经进入编辑器素材：剑馆、静修堂、农家小院、种子农具铺、草药医馆、木作院、书院、临水客栈、马厩、渔舍、汤屋、粮仓。

本轮建筑母版先覆盖江南最关键的一组：蓝染铺、竹编铺、纸坊、茶摊、石桥、书院、灯市。下一批应补：陶坊/青瓷窑、龙泉剑炉、太湖织埠、百工院院落、产业市场、地区牌坊。

### 人物

运行时当前有三套行走表：玩家、商贩/关联人物、游客。NPC 数据已扩展到多个地区和职业，但第一批像素化前先统一三类基础体型、服饰、随身物和色彩：

- 玩家匠人：能采料、行脚、制作，服装要实用。
- 匠人导师/商贩：职业工具更明显，可承载蓝染、竹编、纸坊等变体。
- 游客/货郎：用于街道生活感，轮廓要和导师区分。

## 下一步拆解顺序

1. 以 `jiangnan_full_map_concept_v01.png` 为总谱，标注江南运行地图需要的水路、主街、桥、码头、市集、作坊、书院、灯市、前景植被。
2. 将建筑参考板拆成运行候选：优先重做 `shop_indigo`、`shop_bamboo`、`craft_paper`、`tea_stall`、`arch_bridge`、`gate`。
3. 将人物参考板拆成 3 套 4x4 行走表草图，再决定是否扩展为职业变体。
4. 回到地图编辑器中载入“江南·苏杭水市”和“江南·百工院”样板，按整图母版手工调整落位，保存草稿。
5. 通过同名替换或新版本文件进入 `public/assets/game/**`，再进入 Phaser 运行时验收。
