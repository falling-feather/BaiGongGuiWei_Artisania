# 百工归位 · Artisania

> 以中国非物质文化遗产「手艺工序」为核心的 2D 像素风俯视街道探索 + 经营模拟 Web 游戏。
> 行脚九州、打通商路、采料开工、守艺传承，在「传承 / 市场 / 生活 / 精神」四维间求平衡，
> 由你的名号与抉择写下属于自己的命运结局。

## 技术栈

React 18 · TypeScript 5 · Vite 5 · **Phaser 3**（像素风俯视世界）· Zustand · Vitest · 纯 CSS

## 快速开始

```bash
npm install
npm run dev        # 本地开发（Vite 热更新）
npm run typecheck  # 类型检查
npm test           # 运行测试
npm run build      # 生产构建
```

## 玩法一览

- **像素街道探索**：WASD / 方向键自由行走，走近体验点按 E 交互；`+/-` 或 Shift+滚轮 缩放画面。
- **设置 / 小地图**：左上齐轮或 Esc 唤出设置（可返回主页，存档自留）；右下小地图实时标记交互点与玩家坐标。
- **开局命名**：取一个名号，名字会贯穿剧情称呼（隐藏开发者印记）。
- **采料 → 精炼 → 手艺**：六种采料小游戏取原料，精炼成半成品，进手艺点消耗半成品产出成品。
- **订单闭环**：亲手制作出成品后可「接订单」交付换市场收入，售价随手艺传承品质浮动。
- **大地图 / 商路**：解锁相邻地区、跨区运料，打通供应链。
- **背包 / 成就 / 剧情分支**：数据驱动的库存、成就册与随抉择分叉的叙事节点。
- **个性化结局**：终局报告的尾声呼应你的名号与「守正 / 趋时」抉择走向。

## 架构一句话

纯函数引擎（`src/engine`）+ 注入式内容（`src/data`）+ Zustand 桥接（`src/store`）
+ Phaser 游戏世界（`src/game`）+ 只读渲染 UI（`src/components`）+ 可替换存档（`src/storage`）。
游戏规则只存在于 `src/engine/reducer.ts`；Phaser 只管空间与移动，二者通过 `EventBus` 通信。

当前进度：**地区优先的 2D 像素探索 demo** —— 引擎 + 手艺工序链（采料/精炼/制作）

+ 全局设置 / 地图缩放 / 小地图坐标 + 成品订单交付经济闭环。
